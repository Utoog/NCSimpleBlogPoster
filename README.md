
# NC simple blog poster
A little tool i created for myself to post things on my private blog.
## Using
Before usage be sure to paste your api key to `nc_api` in `config.json` and your profile name in `nc_profile`.  
## Important
If you have a custom domain, be sure to include your **FULL** domain name (like example.xyz) in `nc_profile` and set `custom_domain` to `true`
## Dependencies
- eel
- neocities https://github.com/neocities/python-neocities
- TinyDB
